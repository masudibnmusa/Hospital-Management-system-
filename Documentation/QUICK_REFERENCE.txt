┌───────────────────────────────────────────────────────────────┐
│  FEATURE DETAILS                                              │
└───────────────────────────────────────────────────────────────┘

█ BACKUP DATABASE
  • Creates folder: backups/YYYYMMDD_HHMMSS/
  • Backs up 5 files:
    - patients.dat
    - doctors.dat
    - appointments.dat
    - bills.dat
    - staff.dat
  • Safe: Never overwrites existing backups
  • Use before: Making major changes or deletions

█ RESTORE DATABASE
  • Lists available backups
  • Enter backup folder name to restore
  • ⚠️  WARNING: Overwrites all current data
  • Requires confirmation (y/n)
  • Must restart application after restore
  • Use when: Recovering from data loss or errors

█ EXPORT TO CSV/EXCEL
  • Creates folder: exports/
  • Generates 5 CSV files with timestamp:
    - patients_YYYYMMDD_HHMMSS.csv
    - doctors_YYYYMMDD_HHMMSS.csv
    - appointments_YYYYMMDD_HHMMSS.csv
    - bills_YYYYMMDD_HHMMSS.csv
    - staff_YYYYMMDD_HHMMSS.csv
  • Open in: Excel, Google Sheets, LibreOffice
  • Contains column headers
  • Use for: Reports, analysis, data sharing

█ DELETE ALL DATA
  • ⚠️  CRITICAL OPERATION - CANNOT BE UNDONE!
  • Shows record counts for each data type
  • Requires TWO confirmations (y/n, y/n)
  • Clears ALL records from memory
  • Deletes ALL .dat files
  • Use for: Fresh start, testing, maintenance
  • ⚠️  ALWAYS BACKUP FIRST!

│  TROUBLESHOOTING                                              │
└───────────────────────────────────────────────────────────────┘

Q: Restored data doesn't appear?
A: You must restart the application after restore.

Q: Can't find backup folder?
A: Check the backups/ directory. Folders are named with timestamps
   in format: YYYYMMDD_HHMMSS (e.g., 20250108_105530)

Q: CSV files won't open in Excel?
A: CSV files are in the exports/ directory. If Excel shows garbled
   text, try: Excel → Data → From Text/CSV → Import

Q: Accidentally deleted all data?
A: If you created a backup, use "Restore Database" option.
   If no backup exists, data cannot be recovered.

Q: Export shows "No data to export"?
A: The database is empty. Add some records first.
